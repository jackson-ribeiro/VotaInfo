Projeto de Dados Sobre o TRE-RJ

Este projeto permite consultar locais de votação e zonas eleitorais através de uma API.

## Funcionalidades

- Consultar todos os locais de votação.
- Consultar locais de votação por bairro, município ou zona.
- Consultar zonas eleitorais.

## Instalação

1. Clone o repositório:

   ```bash
   git clone https://github.com/jackson-ribeiro/VotaInfo
   ```
2. Instale as dependências:

   ```bash
   pip install -r requirements.txt
   ```
3. Configure o arquivo .env com a URL da API:

   ```bash
   API_URL=https://apps.tre-rj.jus.br/api-dados-abertos/
   ```
